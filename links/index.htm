Ссылки по теме выпускной работы

    Перечень статей по теме "Цифровая трансформация и оценка зрелости организаций":

  1. Канев В.С., Полетайкин А.Н., Шевцова Ю.В. (2021). "Технология оценивания цифровой зрелости образовательной организации. Часть I"

    Фундаментальная работа по методологии оценки цифровой зрелости на основе 25 показателей, сгруппированных в технико-технологический, когнитивный и личностный факторы.

  2. Шевцова Ю.В. и др. (2023). "Технология оценивания цифровой зрелости образовательной организации. Часть II"

    Развитие методологии с применением нечёткой логики для работы с экспертной субъективностью при оценке цифровой зрелости.

  3. Westerman, G., Bonnet, D., & McAfee, A. (2014). "Leading Digital: Turning Technology into Business Transformation"

    Практико-ориентированный фреймворк цифровой трансформации с акцентом на цифровое лидерство и переосмысление бизнес-процессов.

    4. Перечень статей по теме "Методы поддержки принятия решений и оптимизации":

  4. Hwang, C.L., & Yoon, K. (1981). "Multiple Attribute Decision Making: Methods and Applications"

    Классический учебник по методам многокритериального принятия решений, включая метод TOPSIS, применённый в работе для ранжирования дорожных карт.

  5. Aamodt, A., & Plaza, E. (1994). "Case-Based Reasoning: Foundational Issues, Methodological Variations, and System Approaches"

    Основополагающая статья по методу рассуждений на основе прецедентов (CBR), использованному для генерации трансформационных действий.

  6. Teece, D.J., Peteraf, M., & Leih, S. (2016). "Dynamic Capabilities and Organizational Agility"

    Теория динамических способностей как основа для планирования адаптивных трансформационных стратегий.

    7. Перечень статей по теме "Формальные модели и алгоритмы планирования":

  7. Zadeh, L.A. (1965). "Fuzzy Sets"

    Основа теории нечётких множеств, применённой для работы с неопределённостью в параметрах трансформационной модели.

  8. Poletaikin, A. et al. (2021). "Forming a Indicators System of Digital Maturity..."

    Работа по формированию системы индикаторов цифровой зрелости с атрибутами достаточности и онтологическим подходом.

  9. Канев В.С. и др. (2023). "Методы и модели конструктивной цифровой трансформации..."

    Отчёт о НИР, представляющий комплексный подход к формализации и оптимизации процессов цифровой трансформации.

    10. Перечень практических ресурсов, библиотек и инструментов:

  10. ISO 37120:2018 "Sustainable cities and communities — Indicators for city services and quality of life"

    Международный стандарт по индикаторам развития городов, содержащий методологию оценки, схожую с подходом к оценке цифровой зрелости организаций.

  11. Decision Model and Notation (DMN) Standard

    Стандарт OMG для моделирования бизнес-решений, полезный при проектировании архитектуры системы поддержки принятия решений.

  12. OWL 2 Web Ontology Language

    Стандарт W3C для создания онтологий, использованный при формализации предметной области цифровой трансформации.

  13. Python Programming Language

    Основной язык программирования для реализации алгоритмического ядра системы и веб-интерфейса.

  14. FastAPI Framework

    Современный фреймворк для создания REST API, использованный при реализации веб-интерфейса системы.

  15. jQuery JavaScript Library

    Библиотека для реализации интерактивных элементов веб-интерфейса, включая режим "что-если" анализа.

  16. Chart.js Library

    Библиотека для визуализации результатов (радарные диаграммы, графики сравнения дорожных карт).
